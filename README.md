# 💎 Jewelry E-Commerce System

A full-stack web application for online jewelry shopping and vendor management.  
It allows users to browse, filter, and purchase jewelry, while vendors manage products and inventory.

---

## 🚀 Features

- User & Vendor authentication (Login/Register)
- Product browsing with categories & subcategories
- Advanced filtering (price, metal type, gender)
- Vendor dashboard for product & inventory management
- Secure payment integration (eSewa)
- Order management system
- Product recommendation feature

---

## 🛠️ Tech Stack

**Frontend**
- HTML
- CSS
- JavaScript
- Bootstrap  

**Backend**
- Python (Django)  

**Database**
- SQLite  

**Payment**
- eSewa API  

---

## ⚙️ Modules

**Admin**
- Manage users, vendors, products, and categories  

**Vendor**
- Add, update, delete products  
- Manage inventory  

**User**
- Browse products  
- Add to cart  
- Place orders  

---

## 🔐 Key Implementations

- HMAC SHA-256 for secure payment verification  
- UUID for unique identification  
- Dynamic product filtering using relational queries  

---

## 📌 Objective

To build a secure and user-friendly platform for online jewelry shopping with efficient vendor management.
